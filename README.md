# AviUtl exedit2用スクリプト MIDI2OTOMAD
β版として公開しました。  
MIDIを読み込んで音MAD風のアニメーションを付与するAviUtl exedit2用スクリプトです。  
  
# 【インストール方法】  
全てのファイルをAviUtl exedit2用のScriptフォルダ、あるいはその下位のフォルダにインストールしてください。  
分かりやすい名前のフォルダを用意してそこに保存することを推奨します。  
  
# 【MIDIの用意】  
現状、一度に一つの音符しか再生できません。  
MIDI内で同じ時間に複数の音符が存在する場合、最も新しいものを優先して読み込みます。  
  
そのため、同時間帯に複数の音符が鳴らないようにしつつ各楽器ごとに別々のMIDIを用意してください。  
  
# 【基本的な使い方】  
1.なるべく上のレイヤーに「MIDI2OTOMADペアレントオブジェクト」を置いて、音MAD部分の長さに合わせて伸ばす。  
2.ペアレントオブジェクト内でBPMや拍子を設定する。  
3.M2O〇〇といった名前のカスタムオブジェクトをペアレントオブジェクトより下に置く。  
4.M2Oオブジェクトで、参照するメディアファイルとMIDIファイルの両方を指定する。  
5.M2OオブジェクトにM2Oフィルタ効果を入れてアニメーションを適用する。  
6.M2Oフィルタ効果の項目を設定する。  
7.各オブジェクトごとに3-7を繰り返す。  
  
# 【仕様】  
## MIDI2OTOMADペアレントオブジェクト  
ペアレントオブジェクトは必要な機能を読み込み、読み込んだMIDIを記憶し、子オブジェクトが参照する時間を更新します。  
全ての音MAD用オブジェクトはこのペアレントオブジェクトを参照に時間を計算します。  
そのため、タイムライン内でオブジェクトを動かしてもタイミングがずれることはありません。  
これにより、  
1.全てのフィルタ効果の発動タイミングが同期する  
2.全てのオブジェクトが同じ音楽の再生進捗と同期し、ズレがなくなる  
3.登場・退場のためにオブジェクトの長さや位置を操作してもアニメーションがズレない  
4.手打ちによるミスを防止する  
の全てが両立します。これにより、検証的なアニメーションを下見したり、細かい修正を入れるのが簡単になります。  

不具合が生じた場合はペアレントオブジェクトをシークバーを合わせながら「リセット」を押し、再度押してチェックを外してください。  
  
## M2O子オブジェクト  
M2O〇〇@midi2otomadと表示される子オブジェクトはメディアとMIDIを読み込んでアニメーションを生成する準備をします。  
指定したMIDIは一度のみ読み込みます。アプリを閉じる、MIDI再読み込みにチェックを一瞬入れる、あるいはペアレントをリセットするまで再度読み込まれません。  
動画を読み込んだ場合、MIDIの音符が鳴るたびに動画が最初から再生しなおされます。  
  
## アニメーション  
M2Oフィルタ効果はM2O子オブジェクトから音符の再生状況を読み込み、アニメーションを行います。 
基本、名前にΔとついた項目が時間と共に減衰(だんだん0になる)し、音符が鳴るたびに効果が再度はじめから再生されます。  
  
### アニメーション共通項目  
フィルタ効果の共通の項目として以下があります：  
時間(拍):　　　　　　アニメーションが減衰し切るまでかかる時間を決めます。BPMによって変動します。  
正規化:　　　　　　　アニメーションの長さを読み込んだ音符の長さに合わせます。時間(拍)が1.00の場合、音符が終わると同時にアニメーションが終わります。  
カウントずらし:　　　音符の偶奇を逆に数えたり、アニメーションの開始カウントをずらします。  
イージング:　　　　　簡単なイージングをつけます。  
勢い:　　　　　　　　一部のイージングの強度を変えます。  
減衰:　　　　　　　　デフォルトでONです。ONだと音符が鳴るたびに効果が発動し、時間とともに減衰(だんだん0になる)します。OFFにすると、音符が鳴るたびに効果が消え、時間が立つに連れ戻ってきます。  
交互に減衰と増幅:　音符の再生ごとにアニメーション開始状態と終了状態が反転します。減衰が交互にON/OFFになるのと同じ効果です。  
交互にΔ反転:　　　　音符が鳴るたびに向きや軸などΔとついた項目を-1倍にします。  
参照MIDI上書:　　　別のMIDIを参照してアニメーションを行います。  

### アニメーション一覧  
#### M2O事前実行環境でペアレントオブジェクト化  
WIP.  
オブジェクトをobj.effect()で呼び出した環境でペアレントオブジェクトとして挙動させます。  
  
#### M2O対応化 
WIP.  
MIDIを読み込み、@midi2otomadオブジェクト以外でもフィルタ効果を使える状態にします。  
  
#### M2O仮想バッファに描画して読み込み
指定したサイズの仮想バッファに一度描画し、読み込み直します。  
サイズが0の場合、その時点でのオブジェクトの幅・高さを参照します。
  
#### M2O基本反転  
音符が鳴るたび、基本効果>反転のうちチェックを入れたもののON/OFFを切り替えます。  
  
#### M2O音符再生中にのみ表示  
音符が鳴っている間のみオブジェクトが表示されるようになります。
  
#### M2O基本移動  
音符が鳴ると、指定した座標まで移動し、徐々に戻っていきます。  
  
#### M2O標準描画  
音符が鳴ると、透明度、回転、拡大率、縦横比、透明度などが指定した値変動し、徐々に戻っていきます。  

#### M2O基本拡大  
音符が鳴ると、指定した座標の分拡大・縮小し、徐々に戻っていきます。  
交互反転にチェックが入っている場合、音符が鳴るたびに拡大・縮小を切り替えます。  
  
#### M2Oステップ回転  
音符が鳴ると回転します。分割数の回数音符がなるたびに一回転します。  
  
#### M2Oステップ拡大  
WIP.  
音符がなると拡大します。ステップ数回音符が鳴るともとに戻ります。  
  
#### M2Oラスター  
音符が鳴るとラスターが適用し、徐々に消えていきます。  
交互ラスターにチェックが入っている場合、音符が鳴るたびに縦と横を切り替えます。  
  
#### M2O色調補正  
色調補正を適用します。音符が鳴ると、Δ値分が加算されて徐々に戻っていきます。  
  
#### M2O斜めクリッピング  
斜めクリッピングを適用します。音符が鳴ると、Δ値分が加算されて徐々に戻っていきます。  
幅が0、Δ幅が0以外、そしてアニメーションが完了した際に消えるようになっています。  
  
#### M2Oマスク  
マスクを適用します。音符が鳴ると、Δ値分が加算されて徐々に戻っていきます。
