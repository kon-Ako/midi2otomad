# AviUtl exedit2用スクリプト MIDI2OTOMAD
とりあえず公開した。  
MIDIを読み込んで音MAD風のアニメーションを付与するAviUtl exedit2用スクリプトです。  
  
# 【インストール方法】  
全てのファイルをAviUtl exedit2用のScriptフォルダ、あるいはその下位のフォルダにインストールしてください。  
分かりやすい名前のフォルダを用意して後者を行うことを推奨します。  
  
# 【MIDIの用意】  
現状、一度に一つの音符しか再生できません。  
MIDI内で同じ時間に複数の音符が存在する場合、最も新しいものを優先して読み込みます。  
  
そのため、各楽器のためそれぞれ同時間帯に複数の音符が鳴ることがないMIDIを用意してください。  
  
# 【使い方】  
## ペアレントオブジェクト  
ペアレントオブジェクトは必要な機能を読み込み、読み込んだMIDIを記憶し、子オブジェクトが参照する時間を更新します。  
最初にMIDI2OTOMADペアレントオブジェクトを読み込み、BPMや拍子を入力してください。  
不具合が生じた場合はペアレントオブジェクトが画面に表示してリセットを押してください。  
　　
全ての音MAD用オブジェクトはこのペアレントオブジェクトを参照に時間を計算します。  
そのため、ペアレントオブジェクトが同じフレームに読み込まれている限りオブジェクトを動かしてもタイミングがずれることはありません。  
  
## @midi2otomad子オブジェクト  
@midi2otomadで終わる子オブジェクトはメディアとMIDIを読み込んでアニメーションを生成する準備をします。  
指定したMIDIは一度のみ読み込みます。アプリを閉じる、あるいはペアレントをリセットするまで再度読み込まれません。  
動画を読み込んだ場合、MIDIの音符が鳴るたびに動画が最初から再生しなおされます。  
  
## アニメーション  
@midi2otomadで終わるフィルタ効果は@midi2otomad子オブジェクトから音符の再生状況を読み込み、アニメーションを行います。 
基本、名前にある効果が時間と共に減衰し、音符が鳴るたびに効果が再度はじめから再生されます。  
  
### アニメーション共通項目  
フィルタ効果の共通の項目として以下があります：  
カウントずらし:　音符の偶奇を逆に数えたり、アニメーションの開始カウントをずらします。  
イージング:　　　簡単なイージングをつけます。  
時間(拍):　　　　アニメーションが減衰し切るまでかかる時間を決めます。BPMによって変動します。  
勢い:　　　　　　一部のイージングの強度を変えます。  
正規化:　　　　　アニメーションの長さを読み込んだ音符の長さに合わせます。時間(拍)が1.00の場合、音符が終わると同時にアニメーションが終わります。  
減衰:　　　　　　デフォルトでONです。ONだと音符が鳴るたびに効果が発動し、時間とともに減衰します。OFFにすると、音符が鳴るたびに効果が消え、時間が立つに連れ戻ってきます。  
交互反転:　　　　音符が鳴るたびに、向きや軸などを反転します。  
参照MIDI上書き:　別のMIDIを参照してアニメーションを行います。そのMIDIを読み込む機能がないため、事前に読み込む必要があります。

オブジェクトをobj.effect()で呼び出した環境でペアレントオブジェクトとして挙動させます。  
  
#### @MIDI2音MAD化  
WIP.  
MIDIを読み込み、@midi2otomadオブジェクトでもフィルタ効果を使える状態にします。  
  
#### 音MAD基本反転  
音符が鳴るたび、基本効果>反転のうちチェックを入れたもののON/OFFを切り替えます。  
  
#### 音MAD基本移動  
音符が鳴ると、指定した座標まで移動し、徐々に戻っていきます。  
  
#### 音MAD基本移動  
音符が鳴ると、指定した座標の分移動し、徐々に戻っていきます。  
交互反転にチェックが入っている場合、音符が鳴るたびに移動方向が反転します。  

#### 音MAD基本拡大  
音符が鳴ると、指定した座標の分拡大/縮小し、徐々に戻っていきます。  
交互反転にチェックが入っている場合、音符が鳴るたびに拡大/縮小を切り替えます。  
　　
#### 音MADステップ回転  
音符が鳴ると回転します。分割数回音符が鳴ると一回転するような角度に鳴ってます。　　
　　
####　音MADラスター　　
音符が鳴るとラスターが適用し、徐々に消えていきます。  
交互ラスターにチェックが入っている場合、音符が鳴るたびに縦と横を切り替えます。  
  
#### 仮想バッファに描画して読み込み  
指定したサイズの仮想バッファに一度描画し、読み込み直します。  
サイズが0の場合、その時点でのオブジェクトの幅・高さを参照します。
