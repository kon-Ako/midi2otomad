# AviUtl exedit2用スクリプト MIDI2OTOMAD
とりあえず公開した。  
MIDIを読み込んで音MAD風のアニメーションを付与するAviUtl exedit2用スクリプトです。  
  
# 【インストール方法】  
全てのファイルをAviUtl exedit2用のScriptフォルダ、あるいはその下位のフォルダにインストールしてください。  
分かりやすい名前のフォルダを用意して後者を行うことを推奨します。  
  
# 【MIDIの用意】  
現状、一度に一つの音符しか再生できません。  
MIDI内で同じ時間に複数の音符が存在する場合、最も新しいものを優先して読み込みます。  
  
そのため、各楽器のためそれぞれ同時間帯に複数の音符が鳴ることがないMIDIを用意してください。  
  
# 【使い方】  
## ペアレントオブジェクト  
最初にMIDI2OTOMADペアレントオブジェクトを読み込み、BPMや拍子を入力してください。  
全ての音MAD用オブジェクトはこのペアレントオブジェクトを参照に時間を計算します。  
そのため、ペアレントオブジェクトが同じフレームに読み込まれている限りオブジェクトを動かしてもタイミングがずれることはありません。  
  
## 子オブジェクト
アニメーションを再生するには、〇〇音MAD@midi2otomadというオブジェクトを読み込みます。  
ここで読み込みたいメディアと参照するMIDIを決めてください。MIDIは一度のみ読み込み、アプリを閉じるかペアレントをリセットするまで再度読み込まれません。  
動画を読み込んだ場合、MIDIの音符が鳴るたびに動画が最初から再生しなおされます。  
  
## アニメーション  
追加でアニメーションを足したい場合、@midi2otomadで終わるフィルタ効果を適用します。  
基本的には、記述した効果が時間と共に減衰し、音符が鳴るたびにリセットされます。  
  
## 共通項目  
フィルタ効果の共通の項目として以下があります：  
カウントずらし:　音符の偶奇を逆に数えたり、アニメーションの開始カウントをずらします。  
イージング:　　　簡単なイージングをつけます。  
時間(拍):　　　　アニメーションが減衰し切るまでかかる時間を決めます。BPMによって変動します。  
勢い:　　　　　　一部のイージングの強度を変えます。  
正規化:　　　　　アニメーションの長さを読み込んだ音符の長さに合わせます。時間(拍)が1.00の場合、音符が終わると同時にアニメーションが終わります。  
減衰:　　　　　　デフォルトでONです。ONだと音符が鳴るたびに効果が発動し、時間とともに減衰します。OFFにすると、音符が鳴るたびに効果が消え、時間が立つに連れ戻ってきます。  
交互反転:　　　　音符が鳴るたびに、向きや軸などを反転します。  
参照MIDI上書き:　別のMIDIを参照してアニメーションを行います。そのMIDIを読み込む機能がないため、事前に読み込む必要があります。
